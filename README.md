# Klyra

**Klyra** is a smart parking management web application designed to make parking easier, faster, and more efficient.  
It enables users to find, book, and manage parking spaces in real time using a seamless and intuitive interface.

---

## Features

- **Smart Parking Detection** – Monitors available parking spots using camera feeds and AI-based object detection.  
- **Real-Time Updates** – Instantly view available and occupied parking spaces.  
- **Online Booking & Payment** – Secure, smooth, and easy-to-use booking and payment system.  
- **Admin Dashboard** – Manage users, bookings, and parking areas efficiently.  
- **Live Feed Integration** – Visual insights through live camera monitoring.  
- **Analytics & Reports** – Gain insights from parking trends and usage data.

---

## Tech Stack

**Frontend:**
- HTML, CSS, JavaScript  
- Supabase for authentication and database  
- Responsive design for all screen sizes  

**Backend:**
- Supabase (PostgreSQL, Auth)
